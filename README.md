# Прогноз денежного остатка ATM

## Что делает проект
Прогнозирует денежный остаток на банкоматах на 14 дней вперёд на основе истории снятий, депозитов и календарных признаков.

## Входные данные
- CSV с колонками:
  - date — дата (YYYY-MM-DD)
  - atm_id — ID банкомата
  - withdrawals — сумма снятий
  - deposits — сумма депозитов
  - is_holiday — 0/1, праздничный день
  - is_weekend — 0/1, выходной
  - day_of_week — номер дня недели (0-6)

## Как запустить
pip install -r requirements.txt
python forecast_pipeline.py --input data.csv --model model.pkl


## Выходные данные
CSV с прогнозом по банкоматам и датам:
- atm_id — ID банкомата
- date — дата прогноза
- predicted_balance — прогноз остатка

## Интерпретация и анализ модели
Важнейшие признаки — лаги баланса и потоков, праздничные и выходные дни. SHAP-анализ показывает влияние этих факторов на прогноз.
